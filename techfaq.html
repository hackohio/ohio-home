OHI/O FAQ

What tech resources am I allowed to use/submit for judging?

You may:
‣ Use any hardware, tools, and infrastructure
‣ Use open source and public content; eg. OSU components, Tensorflow
‣ Use proprietary products; eg. APIs, ML models
‣ Mod any hardware or software, without committing piracy

However, you must accurately portray what you have built:
 I created Linux
I used Linux
 This app uses a custom AI model to do <major breakthrough>
We used a mock/external/existing AI model to implement an interesting use case

*kernel contributors and future Turing Awards winners excluded
